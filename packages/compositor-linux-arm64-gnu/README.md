# @remotion/compositor-linux-arm64-gnu
 
Linux ARM64 binary for the Remotion Rust code
 
[![NPM Downloads](https://img.shields.io/npm/dm/@remotion/compositor-linux-arm64-gnu.svg?style=flat&color=black&label=Downloads)](https://npmcharts.com/compare/@remotion/compositor-linux-arm64-gnu?minimal=true)
 
## Installation
 
```bash
npm install @remotion/compositor-linux-arm64-gnu --save-exact
```
 
When installing a Remotion package, make sure to align the version of all `remotion` and `@remotion/*` packages to the same version.
Remove the `^` character from the version number to use the exact version.
